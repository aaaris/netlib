### introduction

> a website for user to download and share books easily on the Internet

一个用于练手的图书资源网站，基于Vue和Gin打造的**前后端分离**项目，拥有以下功能：

- 用户登录注册登出

- 用户上传、下载书本

- 用户收藏书本 

- 用户评论书本
  
- 管理员用户对书本和用户 curd
